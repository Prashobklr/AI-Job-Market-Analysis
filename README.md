# AI Job Market Analysis Dashboard

📌 Project Overview  
This project analyzes AI job market trends using Power BI to uncover insights related to hiring demand, salary trends, top skills, experience levels, and emerging AI technologies. The dashboard provides an interactive and business-focused view of the AI industry job landscape.

---

## 🔄 Flow of Work

Data Source → Power Query → Data Modeling → DAX → Dashboard

---

## ✅ Steps Followed

### Imported Data
Loaded the AI job market dataset into Power BI for analysis and reporting.

### Data Transformation
Cleaned and transformed raw data using Power Query:
- Removed duplicates
- Fixed data types
- Handled missing values
- Standardized columns

### DAX Measures & Calculations
Created multiple DAX measures and calculated columns for KPI tracking and analysis.

### Data Modeling
Built a star schema model by creating relationships between tables for better performance and accurate analysis.

### Dashboard Development
Designed an interactive multi-page dashboard using:
- KPI Cards
- Slicers
- Charts
- Drillthrough Pages
- Dynamic Visualizations

---

## 📊 Key Insights

- Python is the most in-demand AI skill
- Bengaluru has the highest AI job opportunities
- AI salaries grow significantly with experience
- Emerging technologies like LLMs, LangChain, and Vector DB provide higher salary potential

## 📈 Key Metrics & Analysis
